# Security Policy

Please report vulnerabilities by using the [GitHub Vulnerability Reporting
Feature](https://github.com/bugsink/bugsink/security/advisories/new) which is
[documented
here](https://docs.github.com/en/code-security/security-advisories/guidance-on-reporting-and-writing-information-about-vulnerabilities/privately-reporting-a-security-vulnerability)

or privately on security@bugsink.com.
